⬡ V-Coordinator Help

Paste / CSV Format

People and room data can be mixed in the same paste block.

Smith J, 11:00, 13:30
Doe A, 11:00, 11:30, 14:00, 15:30

11:00, , 3, 2
11:30, Taylor C, 4, 1
12:00, Williams D, 2, 3
12:30, Brown E, 5, 2
13:00, Davis F, 3, 1

Person: Name, start, end — multiple time pairs allowed

Room: time, officiator, brothers, sisters

  • Room 1 (11:00 AM) officiator is always AM Shift — leave blank
  • Lines starting with # are ignored
  • Valid times: 11:00 through 16:45 in 15-minute increments

Sheet Tabs

AM and PM show different rooms. Switch tabs to preview each sheet. Both sheets are always included in the export.

Build Schedule

Renders a pixel-perfect preview image of the schedule. Enable Auto to rebuild automatically whenever names change.

Keyboard Shortcuts

  • Ctrl+B / ⌘B — Build Schedule

Export

Downloads the filled .xlsx file with both AM and PM sheets. Use this to print — each sheet prints on a separate page.

Bros Available

The count per room includes the Off row plus any brother whose availability fully covers that room's 30-minute slot.

Unavailable Slots

Gray cells indicate a brother is not available for that room's time slot. A person must cover the full 30 minutes to be considered available.
